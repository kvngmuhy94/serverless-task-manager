# serverless-task-manager
## Overview
The *Serverless SaaS Task Manager* is a cloud-native application designed to help users manage their tasks efficiently.  
It is built with a *serverless architecture*, ensuring scalability, low maintenance, and cost efficiency.

## Tech Stack
- *Frontend:* React, Tailwind CSS, CloudFront, S3
- *Backend:* AWS Lambda, API Gateway, Python/Node.js
- *Database:* DynamoDB
- *Authentication:* AWS Cognito
- *CI/CD:* GitHub Actions
- *Infrastructure as Code:* Terraform

## Features
- User registration and login
- Create, update, delete tasks
- Secure APIs with Cognito JWT authentication
- CI/CD for automated deployment
